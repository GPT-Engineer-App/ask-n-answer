# ask-n-answer

A Q&A style bulletin board app similar to Yahoo Answers, where users can post questions and receive answers from other users. The app should allow for question submission, answer posting, and upvoting or downvoting of answers. It should have a simple, intuitive user interface for easy navigation.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/ask-n-answer.git
cd ask-n-answer
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
